Техническое задание (ТЗ)

Проект:
Web-приложение для бронирования отелей

Цель проекта:
Создать красивое и функциональное web-приложение, позволяющее пользователям бронировать номера в отелях.


1. Функциональные требования

1.1. Пользовательские роли:
    Гость (незарегистрированный пользователь):
        Просмотр доступных отелей и номеров
        Поиск по фильтрам (дата, цена)
        Регистрация и авторизация

    Пользователь (клиент):
        Бронирование отелей
        Поиск по фильтрам (город)
        Отзывы и рейтинги отелей
1.2. Стабильная работа приложения и непоколебимость ошибками.
1.3. Основной функционал
    Поиск и фильтрация
    Поиск по городам, дате

1.4. Бронирование
    Выбор даты заезда/выезда
    Выбор комнаты

1.5. Отзывы и рейтинг
    Возможность оставить отзыв

2. Нефункциональные требования**

2.1. Интерфейс
    Адаптивный дизайн (десктоп, планшет, смартфон)
    Современный и интуитивно понятный UI/UX

2.2. Безопасность
    Шифрование паролей

2.3. Производительность
    Быстрая загрузка страниц
    Кэширование часто используемых данных
    Оптимизация изображений

3. Технологии

Frontend: JavaScript, CSS
Backend: Python (Flask)
БД: Python (SQL Alchemy), SQL
Авторизация: Python (werkzeug)
Хостинг: Glitch
